Logique pour chaque contrôleur

    Pour les Apprenants (Student) :
        Lister tous les apprenants : findMany avec Prisma.
        Récupérer un apprenant par ID : findUnique.
        Ajouter un apprenant : create.
        Modifier un apprenant : update.
        Supprimer un apprenant : delete.

    Pour les Modules (Module) :
        Similaire aux apprenants.
        Validation supplémentaire pour les champs comme duration et price.

    Pour les Inscriptions (Registration) :
        Récupérer toutes les inscriptions.
        Ajouter une inscription en s'assurant que l'ID de l'apprenant et du module sont valides.
        Mettre à jour ou annuler une inscription.
        Supprimer une inscription.

    Pour les Paiements (Payment) :
        Récupérer tous les paiements.
        Ajouter un paiement lié à une inscription, un apprenant et un module.
        Valider que le montant payé correspond à ce qui est attendu.
        Modifier ou supprimer un paiement en cas d'erreur.
